Тема: СТРУКТУРА HTML-ДОКУМЕНТА. ВИБІР ПРЕДМЕТНОЇ ГАЛУЗІ. РОБОТА З ПОСИЛАННЯМИ, ТАБЛИЦЯМИ, ЗОБРАЖЕННЯМИ, СПИСКАМИ в HTML-ДОКУМЕНТІ.

Мета:

  1. Вивчити основи HTML і CSS.
  2. Розробити базову сторінку звітного документу за допомогою HTML.
  3. Застосувати каскадні таблиці стилів для стилізації елементів HTML.
  4. Додати на сторінку різноманітні елементи, такі як посилання, таблиці, форми, зображення, списки та заголовки.
  5. Розмістити створений сайт в Інтернеті, використовуючи сервіс GitHub Pages для його публікації.
  6. Отримати практичні навички роботи з HTML і CSS, а також здобути знання щодо розробки веб-сайтів та їх публікації.
